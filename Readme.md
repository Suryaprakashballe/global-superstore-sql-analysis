# Global Superstore Sales & Profit Analysis

## Project Overview
This project performs end-to-end data analysis using SQL on the Global Superstore dataset.
The goal is to identify sales trends, profit drivers, loss-making categories, and regional performance.

## Tools Used
- MySQL (Data Cleaning & Analysis)
- Tableau (Dashboard & Visualization)

## Key Analysis Performed
- Yearly sales and profit trend
- Category-wise profit and loss analysis
- Discount vs profit relationship
- Product-wise profit and loss
- Ship mode impact on profitability
- Country-wise sales and profit analysis

## Key Insights
- Furniture category shows losses despite high sales
- High discounts negatively impact profitability
- Standard Class shipping contributes the highest profit
- Certain products consistently generate losses
